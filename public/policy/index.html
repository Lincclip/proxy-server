Image Reference Copy 개인정보처리방침

최종 업데이트: 2024년 8월 20일

중요: Image Reference Copy는 사용자의 개인정보를 수집하지 않습니다. 모든 데이터는 사용자의 브라우저 내에서만 처리되며, 외부 서버로 전송되지 않습니다.

1. 수집하는 정보

Image Reference Copy는 다음과 같은 정보를 수집합니다:

  • 이미지 참조 데이터: 사용자가 복사한 이미지의 URL, 페이지 URL, alt 텍스트, 타임스탬프
  • 로컬 저장 데이터: 브라우저의 localStorage에 임시로 저장되는 이미지 참조 정보

2. 데이터 사용 목적

수집된 데이터는 다음 목적으로만 사용됩니다:

  • Figma 플러그인과의 연동을 위한 이미지 참조 정보 제공
  • 사용자가 복사한 이미지의 출처 추적
  • 확장 프로그램 기능의 정상적인 작동

3. 데이터 저장 및 보안

  • 로컬 저장: 모든 데이터는 사용자의 브라우저 localStorage에만 저장됩니다
  • 외부 전송 없음: 수집된 데이터는 외부 서버로 전송되지 않습니다
  • 자동 삭제: 브라우저 캐시 삭제 시 저장된 데이터도 함께 삭제됩니다

4. 데이터 공유

Image Reference Copy는 수집된 데이터를 제3자와 공유하지 않습니다. 모든 데이터는 사용자의 브라우저 내에서만 처리됩니다.

5. 사용자 권리

사용자는 다음과 같은 권리를 가집니다:

  • 브라우저 설정에서 확장 프로그램을 비활성화하여 데이터 수집 중단
  • 브라우저 캐시 삭제를 통한 저장된 데이터 삭제
  • 확장 프로그램 제거를 통한 모든 데이터 삭제

6. 쿠키 및 추적

Image Reference Copy는 쿠키를 사용하지 않으며, 사용자를 추적하지 않습니다.

7. 아동의 개인정보

Image Reference Copy는 13세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보처리방침 변경

이 개인정보처리방침은 필요에 따라 변경될 수 있습니다. 중요한 변경사항이 있을 경우 확장 프로그램 업데이트를 통해 알려드립니다.
